## 簡介

1. 稱號只會在機器人相關回覆中顯示，例如遊戲期間。
2. 稱號可以通過遊戲獲得成就/分數，滿足一定數量後獲得。
3. 暫時無法查看成就的累積/進度。
4. 滿足條件後不會有提示，請透過指令查詢。
5. 滿足條件有機會修改。
