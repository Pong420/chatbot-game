任何在群組輸入【{Initiate[0]}】或【{Initiate[1]}】即會開啟遊戲，並成為遊戲主持人。

#### 主持人指令 ( 於群組使用 )

| 指令                                 | 功能                                             |
| ------------------------------------ | ------------------------------------------------ |
| {Start.join(' / ')}                  |                                                  |
| {End.join(' / ')}                    | 強制結束遊戲，可以用作查看死亡報告               |
| {Survivors}                          | 查詢還活著的玩家                                 |
| {WhoNotVoted}                        | 查詢還未投票的玩家                               |
| {DeathReport}                        | 遊戲完結後，回覆所有玩家死因和死亡時間           |
| {tRegex(PlayerReport, '<玩家名稱>')} | 回覆個別玩家的行動，例子【Pong的報告】遊戲完結後 |
| {NextShort} 或 {Next}                | 查詢是否能進入下一回合                           |
| {Skip}                               | 跳過當前回合                                     |
