## 接受條款

通過與本聊天機器人互動，您同意遵守以下使用條款和條件。如果您不同意這些條款，請停止使用本聊天機器人。

## LINE 機器人功能

部份人可能不熟悉機器人，LINE 允許機器人使用以下功能，其他功能請自行查詢

1. 接收用戶的訊息，不限於文字，包含照片、影片、語音、聯絡資訊、位置資訊、檔案等資訊。
2. 取得用戶的ID ( 不是LINE ID )，暱稱，頭像，狀態。
3. 知道到用戶發送、取消信息，用戶加入、離開機器人所在群組通知。
4. 取得群組ID、名稱、頭像、人數、群組成員基本信息 ( 同第二點 )。
5. 機器人後台是可以查看所在群組的聊天紀錄，似乎最多能看到最近三天的對話。

## LINE 帳號

1. 您在使用某些本服務，例如「狼人殺」遊戲，首次使用會透過LINE取得你的暱稱和用戶ID並儲存，暱稱會用於相關回覆，您可以使用指令更改，更新LINE 的暱稱並不會同步更新。
2. 部份用戶可能無法取得用戶ID，將無法使用這些服務。

## LINE 發送訊息

點擊某些機器人的回覆/連結後，會彈出一個視窗並請求發送訊息的權限，這個權限是可以代替自動發送訊息到當前的聊天室(機器人/群組)，你可以選擇拒絕，然後手動輸入代替，關閉視窗後將無法發送訊息。

2. 您不得利用本服務進行任何違法、欺詐或侵權活動。
3. 您應遵守所有適用的法律和法規。
